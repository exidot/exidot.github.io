公网架设详细教程

  架设公网服务器之前，最好先把单机的跑通，证明你的程序是有用的
1、 创建一个centos服务器，亲测6.5和6.7都可以用，别的版本不保证


2、 用winscp从虚拟机服务端中提取相关文件。服务端可以是单库的，也可以是双库的，5.26G的双库以及457MB的单库都亲测可以用。单库更方便一点，毕竟文件少，又小，如果你的服务器的带宽不够的话，之后上传这些文件可是要花很久很久的
提取的文件有：（1）服务器文件（如果是home/dxf/game这样的话就把home下的文件都拿出来，如果是root/game结构的话，就把root下的文件都拿出来）。（2）usr/local/share里的GeoIP文件夹


3、 备份mysql数据库里的数据，包括表结构之类的，推荐用oracle自己开发的workbench吧，不然用navicat或者sql-front会很麻烦的，做过mysql开发的同学应该知道


4、 修改IP，（1）打开mysql的备份文件，用notepad，没有的用记事本，把192.168.1.200全部替换成你的服务器公网IP。（2）服务器文件里的所有cfg以及dbmw_stat/table/server_config.tbl文件里的192.168.1.200改成你自己的服务器的公网IP，因为cfg很多，所以这里建议写程序改，不容易改错


5、 在你的公网服务器里安装mysql以及开发库代码分别是

 yum install -y mysql-server mysql mysql-deve
 yum install xulrunner.i686

6、 配置mysql。

 service mysqld start
 mysqladmin -uroot password ‘123456’ （设置root的密码）
 mysql -uroot -p
 123456 （输密码，没有回显的，输完回车）
 GRANT ALL PRIVILEGES ON *.* TO IDENTIFIED
 BY'123456' WITH GRANT OPTION;
 Flush privileges;
 Exit

7、 用workbench连接数据库，导入修改过的sql备份文件


8、 用winscp导入修改过的服务器文件到跟你虚拟机里原来位置一样的地方，run所在的那个目录下的所有文件的权限改成0755，可以用winscp改，也可以chmod


9、 关闭防火墙，开机启动mysql

service iptables stop
chkconfig iptables off
chkconfig mysqld on

10、 把GeoIP文件夹放在usr/local/share下


11、 把lib.zip解压后全部导入放在usr/lib下


12、 ./run
