Linux服务器架设DNF教程

  一、系统选择
请选择CentOS 5.6或者5.8的，64位系统
6.4 6.5的问题比较多，官方推荐的是5.8
二、系统服务精简
删除不必要的服务和软件，可参考http://www.laozuo.org/1917.html
注：千万不要删除或者停止sshd服务！！！！
三、DNF服务端文件准备
1、虚拟机中的dxf文件夹，即服务端的根目录
2、GeoIP-1.4.8
3、虚拟机中/usr/lib 里面的libGeoIP.so  libGeoIP.so.1 libGeoIP.so.1.4.8三个文件
4、mysql文件夹，具体在/var/lib/mysql
怎么上传到服务器应该不用说了吧，保证服务器目录结构和虚拟机中的一样
四、安装
1、安装数据库
yum -y install mysql mysql-server
chkconfig mysqld on
覆盖mysql配置文件
cp /usr/share/mysql/my-medium.cnf /etc/my.cnf 

 

在my.cnf的[mysqld]中添加两行
[mysqld]
port            = 3306
datadir=/var/lib/mysql
user=mysql
socket          = /var/lib/mysql/mysql.sock
skip-locking
key_buffer = 16M
max_allowed_packet = 1M
table_cache = 64
sort_buffer_size = 512K
net_buffer_length = 8K
read_buffer_size = 256K
read_rnd_buffer_size = 512K
myisam_sort_buffer_size = 8M

启动mysql
service mysqld start
2、准备库文件
将libGeoIP.so  libGeoIP.so.1 libGeoIP.so.1.4.8 分别拷贝到/usr/lib/ 和 /lib/security/
将/home/dxf/game 下面的 libantisvrimport.so libnxencryption.so 拷贝到
/usr/lib/ 和 /lib/security/
3、安装GeoIP
yum -y install gcc gcc-c++ make zlib-devel
进入GeoIP-1.4.8目录
cd /home/dxf/GeoIP-1.4.8
执行
./configure
make
make check
make install


4、替换IP，启动dnf


cd /home/dxf

sed -i "s/你在虚拟机里面用到的ip/你的外网ip/g"  `find . -type f -name "*.tbl"`

sed -i "s/你在虚拟机里面用到的ip/你的外网ip/g"  `find . -type f -name "*.cfg"`
./run

 


红色的是命令，最好直接复制不要自己打！


完毕！祝大家成功！
