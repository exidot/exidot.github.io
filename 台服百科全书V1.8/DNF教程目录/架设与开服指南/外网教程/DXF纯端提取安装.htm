DXF纯端提取安装

  dxf纯端架设步骤

服务端：

1.安装linux，推荐centos6.5或者6.7的x64版本
2.安装32位的开发库
yum install xulrunner.i686
3.安装mysql
yum install -y mysql-server mysql mysql-deve
service mysqld start
然后停止mysql导入数据
service mysqld stop
（才有直接覆盖文件的方式，无需mysql去导入sql）

4.把mysql数据覆盖到/var/lib/mysql,给予文件夹777权限，然后启动
service mysqld start
设置mysql开机启动
chkconfig mysqld on

5.给予root文件夹777权限，输入命令./ar启动服务端
（服务端我喜欢丢到root文件夹里面）

上传文件注意2个地方：

1.把game下面的libnxencryption.so复制一份到lib文件夹
2.usr\local\share\GeoIP\GeoIP.dat 记得这文件，放入相同位置


很多人说单数据库有问题，那是数据库合并不完善，可以直接导入70台服数据库，即可解决
