正则表达式二（分组替换）

——by：lblb699 原帖地址：http://www.pvfba.com/forum.php?mod=viewthread&tid=360&extra=page%3D1

 

 

上回说到正则区块搜索，批量替换的方法，这回我再给大家说说正则分组替换的方法，正则是很强大的工具，我懂的也只是皮毛，而且写一次还得查大量资料， 话说这次也是压杆的议题，接下来进入正题：
有这么一段代码：  
27044        2        5        6        10102        0        0        0        0        0        0        1        7486        `百中選一的青銅劍`
要替换成：
27044        2        5        6        10102        0        0        0        0        0        0        1        7486        <3::name_27044``>

如果文件中只有这么一行的话， 当然手动去更改也可以，但如果出现几千行类似的代码，手动去改的话……
我们现在来分析一下：
我们要改动的地方是将`百中選一的青銅劍`变为<3::name_27044``>，而<3::name_XXX``>中的XXX是每行首个制表符之前的数字，中间的一段代码不变，要实现这种替换，就需要用到分组的方式，我们来把代码分为三段：
（27044）    （2        5        6        10102        0        0        0        0        0        0        1        7486）    （`百中選一的青銅劍`）
接下来用正则将三段代码分隔并转义：(\w+)\t(\w+.*)`(\w+.*)`
颜色代表相应的分组，其中\t代表第一个制表符。
接下来我们写替换的正则表达式：(\1)\t(\2)(<3::name\_\1``>)
每一段分组都用括号区别，\w+可以理解为一段变量，无字数限制，在搜索部分出现几次，那么替换部分就可以用\1,\2,\3……代表相应的分组。
