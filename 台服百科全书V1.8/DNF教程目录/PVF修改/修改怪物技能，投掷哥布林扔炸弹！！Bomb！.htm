打开投掷哥布林的文件

[throw attack]
'[attack index]' 0 '[passive object number]' 1 '[passive object filename]' 'Particle/ThrowStone.ptl' '[object type]' 1 '
[passive object index]' 30001 '[power]' 0 '[throw frame]' 2 '[passive object start x cood]' 20 '[passive object start y cood]' 0 '
[passive object start z cood]' 60 '[attack number]' 1 '[throw sound]' 'THROW_STONE'
[/throw attack]
找到这段代码，投掷攻击，30001，这个数字是投掷石头的代码，passive object filename目标文件名，ThrowStone.ptl 这个文件就是投掷技能，需要把替换为走狗的投掷技能

打开这个看到8225的代码就是走狗的炸弹代码了，把30001改为8225
接下来要做的是，找到走狗投掷炸弹的技能，也就是.ptl文件在走狗怪物文件里，至于怎么找8225这个代码

 搜索8225

8225 'ActionObject/Monster/Jeff/Jeff_FrontMissile.obj'
Jeff_FrontMissile.obj' 记住段英文

这个.ptl就是投掷炸弹的技能，随便复制到一个地方重命名为 throwstone 然后覆盖到以下位置去

然后写入这个ptl就完成了
投掷哥布林的文件也要写入，因为代码30001改为了8225
