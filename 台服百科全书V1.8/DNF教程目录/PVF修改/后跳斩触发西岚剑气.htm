后跳斩触发西岚剑气

——by：黄色哥_

首先研究西岚的代码发现关键的文件在key这个文件夹

 

打开发现了很多.key文件比如quadruple_backstepcutter.key打开它发现

里面真是特效代码，可以发现20065正式剑气代码
但是你把20065这个代码替换后会发现，剑气不会动，
 尝试了很多次都没办法解决这个问题，终于在光炎剑特效代码上面发现了一些细节
8085 0 30 0 0 0"draongheadenergyball.ptl" 发现后面有一个。draongheadenergyball.ptl文件，这个就是关键
在key文件里面发现[PARTICLE FILENAME]
'key/grandwave.ptl' 刚好也有一个ptl文件，打开发现里面就是记录移动的数据

所以想让剑气移动就得解决ptl这个文件
因此我就想到了一个办法，那就是替换现有的ptl文件，我把光炎剑特效配置文件draongheadenergyball.ptl的内容替换成grandwave.ptl。在把代码改成20065 0 0 0 0 0 'DragonHeadEnergyBall.ptl'
写入，测试发现终于成功了


相信大家因该看懂了吧？总结就是知道剑气代码后，还得知道ptl文件，我用光炎剑的ptl替换的，你们也可以换其他的替换。举一反三，阿甘左的同理，大家可以去试一试

[if]


[use skill]
'swordman' 49
[/use skill]


[cooltime]
100
[/if]


[then]


[target]
'all enemy' -1
[passive object]
20065 0 0 0 0 0 'Straight3.ptl'
[/passive object]


[/then]
完整代码，经大神点拨，直接就可以调用自带的Straight3.ptl文件了，不用替换什么的
