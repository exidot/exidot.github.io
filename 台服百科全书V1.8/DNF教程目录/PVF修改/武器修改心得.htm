武器修改心得

——by：下落狂风

首先，工具是需要的
你需要Notepad++，用来打开文件进行修改，
Decrypt，用来解压pvf和导入修改的文件，
当然最好是有网络咯。

如果有其他软件有相同功能也可以代替。

首先要用Decrypt解压pvf

由于解压后这东西的庞大以及后续删除非常麻烦，还望三思而行

解压后，如何去寻找你想要修改的武器呢？

在解压的其中的equipment文件夹里面，有着equipment.kor.str总物品文字描述
equipment.lst 总物品路径描述

用Notepad++打开，ctrl+F就可以查找相关的武器路径
然后就可以把对应文件用Notepad++打开了

在这里，选择的装备也是有讲究的，因为我并没有专业版，可以凭空创造一个文件加进去
所以只能在原有的基础上进行替换，那我们希望要什么类型的武器呢？

个人认为，最好的是有

name 物品第一个名字
name2 代码名字之类的 不需要修改
basic explain 基础描述
detail explain 详细描述 在游戏里按键可以展开的
flavor text 物品最低端的灰色文字描述

为什么说这把武器很好呢？
第一，该有的都有了
进行修改很方便
尤其是是flavor text ，自带的，可以修改文字用来装逼
我尝试过给没有flavor text 的武器自行添加，均失败告终
如果有成功的人还望指点一二。
第二，武器是古老的，已经属于不会再爆出来的了
但是可以发送邮件给自己，降低了其他的风险性。
当然了，我是单机玩家，所以无所谓了

因为这个是总索引，所以如果在这里修改了，然后用Dec写入了pvf，那么这武器的描述就会变了。
也就不需要在武器那边修改了

解压了pvf之后，比如，我们想修改古老的天脊乾坤剑
怎么搜索呢？
equipment.kor.str总物品文字描述 里面搜索天脊乾坤劍
注意这里是繁体,搜索到物品代码是28265

然后在equipment.lst 总物品路径描述里面搜索28265
搜索到28265 'character/swordman/weapon/beamsword/n_beamswd_hellohella.equ'
这就是它的位置和文件名,对着点过去即可.

插播着重讲一个编写的技巧！！！

首先一定要备份确认无误的原文件或者半成品文件！

编写武器，不要一次性全部写完再保存，没有谁能够做到完全无错的

写一部分 比如从grade写到 icon上面 点保存，然后用Dec导入，如果成功，立刻导出
比较一下前后文件的大小，如果从kb变成了几百b，那么必然有代码写错了
这个时候就把备份的覆盖进来 重新写过
如果1.51kb变成了1.52kb 也要打开看一下，是否哪个数字后面多了一个0什么的
一般情况是不会有问题的 但是不排除会有这样的情况发生 那必须把这个0删除

举例
[elemental property]
'[water element]'
表示武器冰属性攻击
有时候这0.01kb多了一个0变成了
[elemental property]
'[water element]' 0
这样的 那必须删除这个多余的0 同时保证这一行最后有个空格再回车
然后再次保存再次写入
如果失败 从备份里面覆盖过来

所以备份一定要时刻做！

写了个半成品 导入成功 也导出成功没变化
最好先备份一份原来的pvf

然后可以替换pvf，进入游戏，创建一个1级小号，用发邮件的形式看看能否打开邮件
如果点开邮件瞬间未响应，不要慌，关了游戏，清除邮件，回去继续修改武器吧

循环往复直到最后成功

装备的属性尽量多元化，尽量有自己的特色，尽量不要太变态

差一点没关系的，已经比国服原版牛逼多了，技术在哪怎么会打不过呢

各种装备的特色要自己去找
比如想加跳跃力，找到墨竹手镯或者苍穹之影，具体位置上文已经讲了
如果不知道名字的，所以喊你要有网，可以百度加跳跃力的首饰
然后重复上面的步骤

想要近距离攻击力 搜索天选之剑-格朗，诸如此类的要去参考其他武器的

最好把这些素材全部复制一遍，然后放在桌面新建一个文件夹，改一下文件名
这样用Note打开也直观明了。

左边只放一个要修改的武器，右边都是素材，最好改一下名字，不要在原文件改名字哦！

内容
[if]
内容
[/if]
空行
[then]
内容
[/then]
空行
上面这个是触发效果的标准格式 if 开始 if结束 then(这个时候)会有什么什么然后结束(/then)
第一个内容表示可以挨着其他文字，第二个内容和第三个内容 复制武器特效的时候就有了
空行表示必须空行 不管下面是什么 都要空行否则就写不进去Dec的
if里面的很多 对照的武器属性是可以猜出来的
[duration] 持续时间 [probability]概率等等 这个是硬性最好不变 修改下面的文字即可

不熟练的时候，改完一个if then 就保存导入 同时导出验证
如果OK，替换pvf进入游戏打开邮件看看效果，但不要拿出来直到完全改好
或者拿出来看看，然后销毁继续修改下一个if then 然后重复

当你的武器、称号修改成功后，找到最初没有修改的pvf
把对应的索引文件、修改好的武器导入，然后替换本地DOF里面的pvf
这样可以尽量减小pvf的体积，反复用同一个pvf修改多次是会体积越来越大的
所以我们要保证次数越少越好

如果学习了NPK的制作方法，就可以做好放入imagepack2 跟模型替换一样
这个我还没有去看就不会了
有兴趣的可以去看看

最后讲一个实例吧
如何复原原有的战神天袭手套连击伤害加成的效果

左边第一个为战神套装胜点引发的效果
stat change 当XX状态为
VP ＞ +10000 胜点大于10000的时候
VP＜ +11001 胜点小于11001的时候
+ 表示是正数
可以合起来翻译 10000＜VP 即胜点 ＜ 11001
触发的效果是追加伤害1%

中间的if为战神套连击数引发的效果
冷却时间为20秒
连击数到达100的时候 触发(then)效果为持续20秒的攻速、释放加成10%
因为我们想要的是连击数加成伤害
只需要把左边的vp换成combo 胜点换成连击数
然后修改对应加成伤害 以及连击数即可 原来是胜点数
然后就变成了最右边的成品 连击数在20-50之内 伤害追加5%
之后的继续换即可

关于装备说明同步的问题
修改了总文字描述，保存了，放进了pvf了
就即时生效
即使你是用文件名不一致的文件写入，一样会变成总文字描述的武器说明

如果不放心，可以在总文字描述输入pvf之后，把要修改的武器也输出一遍

这个是移动速度，攻击速度，释放速度的写法，30表示加成值为3%

另外我发现骨戒、古老的骨戒、克罗之戒在很多版本里面都被删除了效果

图为对龙族伤害加成12% 恶魔族伤害加成23% 破招攻击力加成25%
分别可以在雪之荣誉、嗜灵之戒、黑色瘟疫系列腰带里面找到
可以看到第一段和第二段除去伤害变化之外 区别只有一个 dragon 龙 和 devil 恶魔 的区别
