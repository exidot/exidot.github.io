模块相关

装备在某些地图中触发进行限制的模块：

[module]
'[dungeon]' '[tournament dungeon]' '[dead tower]' '[war room]' '[blood system ]'
[/module]

  • 普通地图：dungeon
  • 地下城类型：dungeon type
  • 锦标赛地图：tournament dungeon
  • 死亡之塔系列地图：dead tower
  • 战场类地图：war room
  • （疑似无尽的祭坛类地图）：blood system 【理由：无尽祭坛有一个blood point系统。】
  • 怪物攻城地图：assault
  • 竞技场：pvp
